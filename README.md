## 📌 Overview
This project analyzes customer behavior and revenue patterns using a real-world e-commerce dataset.  
An interactive web application was built using Streamlit to explore trends, customer segments, and business insights.

---

## 🚀 Features
- 📊 Revenue and KPI analysis  
- 📈 Sales trend visualization  
- 👤 Customer segmentation using RFM  
- 🛒 Product category performance analysis  
- 🎛️ Interactive Streamlit dashboard  

---

## 🧠 Key Insights
- Top customers contribute a significant portion of total revenue  
- Certain product categories dominate sales performance  
- Repeat customers generate higher average order value  
- Opportunity exists to improve customer retention  

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)  
- Data Visualization (Matplotlib, Seaborn)  
- Streamlit (Interactive Web App)
